# 홈 여행 리스트 페이지 설계

## 목표

접속 시 여행 목록을 카드 형식으로 보여주는 홈 페이지를 추가한다.
기존 제주도 태교여행 플랜 페이지는 그대로 유지하되, 홈에서 카드 클릭 시 진입할 수 있도록 연결한다.

## 변경 사항

### 파일 구조
- `public/index.html` → `public/trip.html` (기존 여행 플랜 페이지, 경로만 변경)
- `public/index.html` (신규) → 여행 목록 홈 페이지
- `public/js/app.js` → 경로 변경에 따른 영향 없음 (독립적으로 동작)
- `public/css/style.css` → 기존 파일 유지, 홈 스타일은 `public/css/home.css` 신규 추가

### 홈 페이지 (`public/index.html`)

**헤더**
- 타이틀: "담패밀리 여행 플랜 ✈️"
- 배경 흰색 카드, 하단 그림자

**여행 카드 그리드**
- 2열 그리드
- 카드 1: 제주도 태교여행
  - 썸네일: 연두빛 그라데이션 + 🌴
  - 제목: "제주도 태교여행 with 도담🩵🐕"
  - 날짜: 26.05.28 ~ 26.05.31
  - 배지: 계획중 (파란색)
  - 클릭 시 `/trip.html` 이동
- 카드 2: 새 여행 추가하기 (점선 빈 슬롯, 비활성 스타일)

**스타일 원칙**
- 배경: #f8f9fa (연한 회색)
- 카드: 흰색, border-radius 16px, 박스 섀도우
- 호버 시 카드 살짝 위로 떠오르는 효과
- 폰트: 시스템 폰트

### 서버 변경 없음
- Express의 `express.static` 이 그대로 `/trip.html` 을 서빙함
- API 경로 변경 없음
